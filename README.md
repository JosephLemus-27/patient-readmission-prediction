# Predicción de Hospitalización de Pacientes

Proyecto de aprendizaje supervisado desarrollado para predecir si un paciente será hospitalizado utilizando información clínica, demográfica y antecedentes médicos.

## Descripción

Este proyecto aplica técnicas de Machine Learning para identificar pacientes con riesgo de hospitalización a partir de variables como edad, índice de masa corporal, presión arterial, niveles de glucosa, colesterol, antecedentes médicos y hábitos de salud.

La metodología utilizada fue CRISP-DM, siguiendo las etapas de comprensión del negocio, comprensión de los datos, preparación de los datos, modelado, evaluación y despliegue.

## Tecnologías utilizadas

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Modelos evaluados

* Regresión Logística
* Random Forest
